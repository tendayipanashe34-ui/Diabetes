About this system

This platform implements the Deep ANN (5-layer) and TabTransformer models from the clinical diabetes dataset notebook. Features used: chol, stab.glu, hdl, ratio, age, height, weight, waist, hip, bp.1s, bp.1d, time.ppn + gender.

Target variable: diabetes = 1 when glyhb ≥ 7, else 0. Dataset: 4,416 patients · 111 diabetic · 4,305 non-diabetic. All predictions are saved to SQLite database with full audit trail.
